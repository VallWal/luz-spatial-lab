# LUZ Property Passport — Founding Walk Prototype

A high-fidelity clickable UX prototype of the Property Passport Founding Walk, built for LUZ Spatial Lab.
Mock data only — no backend, no camera, no NFC, no AI services, and no connection to the production LUZ Inspector app.

```bash
npm install
npm run dev
```

Open the printed URL on a desktop browser, or on an iPhone on the same network for the intended experience.

Full documentation: [`../docs/prototype-guide.md`](../docs/prototype-guide.md)
